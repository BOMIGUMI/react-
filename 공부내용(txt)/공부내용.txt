- react : 사용자 인터페이스를 만들기 위한 JS 라이브러리
	ㄴ 사용자 정의 태그를 만들기 위한 JS 라이브러리

- 사용자에게 서비스를 보여주고 싶음 -> html 
	ㄴ 즉, react란 사용자에게 좀더 효율적으로 보여주기 위한것 -> 웹 페이지를 좀더 세분화 시켜서 보여주는것
	ㄴ 필요한 페이지에 대해서 컨포넌트를 만들어주어 필요할 때마다 Top 태그를 통해 고정될 부분을 정하고
		고정된부분을 쓰고 싶을 때 마다 컴포넌트의 이름을 명시해 주기만 하면 편리하게 쓸 수 있음

- 컴포넌트 : 사용자 정의 태그

- 터미널에 설치해야 할 것들
	ㄴ npx create-react-app . => 현재 구조에(최종 경로에) react 프로젝트 구성 만들기, . 에다가 폴더명 적어주어도 됨! -> 지정해준 폴더명에 구성만들기
		ㄴ Happy hacking! 이라는 글씨가 뜨면 완료!
	ㄴ npm start => 현재 만든 프로젝트를 시작하겠다(실행겠다)라는 명령어. -> 우리가 만들 웹 페이지가 나옴
